# Portfolio v1

This is the first version of my developer portfolio.

## Technology Used

- React.js
- Vite
- CSS


Follow these steps to run the project locally:

```bash
git clone git@github.com:mukuls1107/portfolio-v1.git
cd portfolio-v1
npm install
npm run dev
```
